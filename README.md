## COMO INICIALIZAR EL PROYECTO

### PARA EL BACKEND.

1. Para usar el servidor hay que tener instalado nodejs y mongoDB.
2. Pasos para iniciar el servidor:
- cd backend
- npm install
- npm run dev

### PARA EL FRONTEND

1. Pasos para iniciar el frontend:
- cd frontend
- npm install
- npm start

## Tecnologias usadas

* Node js
* Express
* Mongodb

- *FRONTEND*:

* React
